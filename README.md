# Portfolio UI

This project is built using `react` & TypeScript with `create-react-app`, running on Node 10.16.16 LTS.

# Tools

UI are built with `react`, styling are handled with `sass`. 
Other UI libraries includes `react-bootstrap` and `font-awesome-icons`, `framer-motion`.
State managment are handled using the bog-standard `useState` (good enough).
For making HTTP requests, `axios` are used.

# Other informations

On development, the default port is **3000**.
